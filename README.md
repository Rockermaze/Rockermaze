## Projects

### YourPOS

**Modern Mobile Point of Sale System**

A cross-platform POS application for retail stores and small businesses. YourPOS focuses on fast billing, inventory control, barcode and QR scanning, customer and supplier management, offline-first local storage, and real-time synchronization with a backend server.

**Highlights**

- Flutter-based mobile application
- Inventory and stock management
- Sales and billing workflow
- Barcode and QR scanner
- Customer and supplier management
- Sales analytics and reports
- Backend API integration
- Offline data synchronization
- Local database support

**Tech Stack:** Flutter, FastAPI, Python, SQLite, MongoDB, REST APIs

---

### College Notes Sharing

**Student Notes Sharing Platform**

A web application that helps students upload, browse, download, and share academic notes. The platform organizes study material by department, semester, and subject, making collaboration easier for students.

**Highlights**

- User-friendly interface
- Upload and download notes
- Department-wise organization
- Semester and subject filtering
- Simple authentication workflow

**Tech Stack:** PHP, HTML, CSS, JavaScript, MySQL

### AI Voice Assistant

**Offline AI-Powered Desktop Voice Assistant**

An intelligent desktop assistant that understands natural language commands and executes system operations. It uses a modular architecture with speech recognition, LLM-based intent understanding, and command routing for fast local automation.

**Highlights**

- Speech-to-text input
- LLM-based intent recognition
- Natural language command execution
- Windows automation
- Open and close applications
- Volume and brightness control
- System control for shutdown, restart, and sleep
- Modular command architecture
- Fast command routing pipeline

**Tech Stack:** Python, FastAPI, WebSockets, Ollama, LLMs, Vosk, Windows APIs
