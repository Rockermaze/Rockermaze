### YourPOS

> Modern mobile point-of-sale system for retail and small businesses.

Cross-platform POS application focused on fast billing, inventory control, barcode and QR scanning, supplier management, customer management, offline-first storage, and backend synchronization.

**Highlights**

- Flutter-based mobile application
- Sales, billing, inventory, and stock workflows
- Barcode and QR scanner support
- Customer and supplier management
- Offline-first local database support with backend API integration
- Sales analytics and reporting foundation

**Tech Stack:** Flutter, FastAPI, Python, SQLite, MongoDB, REST APIs

---

### College Notes Sharing

> Student notes sharing platform for academic collaboration.

A web application that helps students upload, browse, download, and organize academic notes by department, semester, and subject so study material is easier to discover and share.

**Highlights**

- Upload and download notes
- Department-wise organization
- Semester and subject filtering
- Simple authentication workflow
- Clean student-friendly browsing experience

**Tech Stack:** PHP, HTML, CSS, JavaScript, MySQL

---

### AI Voice Assistant

> Offline AI-powered desktop assistant for Windows automation.

An intelligent assistant designed around speech recognition, LLM-based intent understanding, command routing, and local system control. The project uses a modular architecture so commands can be extended cleanly.

**Highlights**

- Speech-to-text command input
- LLM-based intent recognition
- Natural language command execution
- Open and close applications
- Volume, brightness, shutdown, restart, and sleep controls
- Fast command routing pipeline

**Tech Stack:** Python, FastAPI, WebSockets, Ollama, LLMs, Vosk, Windows APIs

### UAreAble-FE

A Flutter application built to make sign language education more accessible through interactive lessons, visual demonstrations, quizzes, practice exercises, and progress tracking.

**Highlights**

- Practice exercises and quizzes
- Progress tracking
- Clean responsive mobile UI
- Cross-platform Flutter foundation

**Tech Stack:** Flutter, Firebase, Dart
